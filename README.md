# re-alm-template

A Leiningen template for [re-alm](https://github.com/vbedegi/re-alm)

## Usage

```
lein new re-alm <project-name>
```

To enable webpack, use `+webpack` flag:

```
lein new re-alm <project-name> +webpack
```
